## Acceptance criteria not testable automatically

- **Icon visual quality** — whether the icon "visually represents the Stex/Draftable brand" and "renders crisply" at 16x16, 24x24, 32x32 requires manual visual inspection in the Windows system tray
- **Icon format correctness** — the ICO file was validated structurally (valid header, 3 PNG payloads at correct sizes) but rendering correctness requires an Electron runtime
