## What each test covers

| Test | What it covers |
|------|---------------|
| calls closeRequestHandler instead of hiding directly when handler is set | When `setOverlayCloseHandler` has been called with a handler, the WINDOW_HIDE IPC routes through that handler instead of calling `hideOverlay()` directly.
