# Test Notes

## Test files created/modified

- `src/main/soniox.test.ts` — added `describe('endpoint marker filtering')` block with 4 tests

## What each test covers

1. **Filters `<end>` token from final tokens** — mixed final batch with `<end>` and normal token; asserts only normal token is forwarded
2. **Filters `<end>` token from non-final tokens** — mixed non-final batch with `<end>` and normal token; asserts only normal token is forwarded
3. **Does not filter tokens with similar but different text** — tokens like `<br>`, `end`, `<END>` pass through unchanged
4. **Response with only `<end>` token emits neither callback** — pure `<end>` response triggers no token callbacks
