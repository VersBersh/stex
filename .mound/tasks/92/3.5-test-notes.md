# Test Notes

## Test files created

## What each test covers

- **initLogger**: creates log directory, closes previous stream on re-init, graceful fallback on failure
- **log levels**: filters by configured level, default to info, debug shows all
- **format**: timestamp/level/message structure, `%s`/`%d` format string interpolation, extra args
- **console mirroring**: debug/info → console.log, warn/error → console.error
- **file rotation**: rotates when file exceeds 5 MB, no rotation when under limit
